编辑小提示

发表于2023-06-28|更新于2025-09-17
|浏览量:

所有hexo命令前加上npx
gitbash中的复制粘贴是ctrl+install shift+install
改完元素后记得
npx hexo clean npx hexo g
本地测试效果用npx hexo s 随后用localhost:4000来连接
上传到网络是npx hexo d
接下来进一步升级网页
三部曲改为
hexo clean hexo g gulp hexo d
来优化网页
